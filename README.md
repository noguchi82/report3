# report3
![test](https://github.com/noguchi82/report3/actions/workflows/test.yml/badge.svg)
千葉工業大学　ロボットシステム学　課題3のリポジトリである。

## 概要
入力された整数が奇数であるか、偶数であるか判定する。

## インストール方法
$ git clone https://github.com/noguchi82/report3.git

$ cd report3

## 実行方法
任意の整数を入力し

$ ./result

もしくは

$ echo 数字｜./result

で実行する。

## 例1
$ echo 3 | ./result

奇数 

## 例2
$ ./result

3

奇数

## 必要なソフトウェア
Python 3.7～3.10

## 動作確認済環境
Ubuntu 22.04.1 LTS
